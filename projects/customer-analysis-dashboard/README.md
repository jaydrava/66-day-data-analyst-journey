# Customer Analysis Dashboard

## Project Overview
Interactive Tableau dashboard analyzing customer purchasing behavior, segmentation, and lifetime value for strategic sales decisions.

## Business Problem
Sales team needed visibility into:
- Most valuable customers
- Purchase patterns and repeat behavior
- Customer segmentation insights
- Regional performance by segment

## Dataset
- **Source:** Customer transactions (2024)
- **Size:** 150-200 transactions
- **Columns:** Transaction ID, Customer details, Product info, Purchase date, Amount

- **Analytics:** Customer segmentation, trend analysis, comparative metrics

- Customer value segmentation
